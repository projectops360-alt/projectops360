# regressions/

Investigation files for individual regressions. The canonical log lives in
[`../10-regression-log.md`](../10-regression-log.md); create a file here (`REG-0XX-name.md`) when
a regression needs a full investigation trail (evidence, git archaeology, repro steps,
resolution).

**Each investigation should cover:** description, observed vs expected, evidence (commits,
branches, files), root cause, impact, fix/decision, verification, and final status.
